创建cc环境


安装 npm 和 node： 
========================
curl -fsSL https://deb.nodesource.com/setup_lts.x | sudo -E bash -
sudo apt-get install -y nodejs
node -v
npm -v

注：npm会在安装nodejs时自动被安装，所以安装nodejs之后可以直接查看npm的安装信息。


安装最新版claudecode
=================================
# 检查node版本，需 >= 18.0.0
node -v  

# 安装claudecode最新版：
npm install -g @anthropic-ai/claude-code@latest

# 验证版本
claude --version
